# HELM Peptide Sequence Generator

This repository contains two scripts (Python) to generate random peptide sequences in HELM 2.0 format.

## Requirements
* **Python**: 3.x

## Usage

### Python Script
Run the script with optional flags for length (`-l`) and number of sequences (`-n`).
python helm_maker.py -l 15 -n 100 -o my_peptides.csv
